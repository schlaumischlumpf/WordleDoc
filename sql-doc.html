SQL Documentation

This document provides a comprehensive overview of SQL commands and features.

Basic SQL Commands

SELECT

SELECT * FROM table_name;

INSERT

INSERT INTO table_name (column1, column2) VALUES (value1, value2);

UPDATE

UPDATE table_name SET column1 = value1 WHERE condition;

DELETE

DELETE FROM table_name WHERE condition;

Advanced Features

SQL supports many advanced features such as joins, subqueries, and indexes.

JOIN

SELECT * FROM table1 JOIN table2 ON table1.id = table2.id;

Subquery

SELECT column_name FROM table_name WHERE column_name IN (SELECT column_name FROM another_table);
