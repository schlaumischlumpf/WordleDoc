1 Einleitung

Die Spielmechanik von Wordle basiert auf einem ausgeklügelten Algorithmus zur Übereinstimmungsprüfung...

2 Prüfalgorithmus

Der Prüfalgorithmus besteht aus mehreren Schritten, um die Übereinstimmung zwischen dem eingegebenen Wort und dem Zielwort zu überprüfen...

Schritt 1: Umwandlung in Großbuchstaben

Eingabe normalisieren

input = input.toUpperCase();

Schritt 2: Vergleich mit Wörtern aus der Wortliste

Wort-Validierung

for (int i = 0; i < WOERTER.length; i++) {
    if (input.equals(WOERTER[i].toUpperCase())) {
        return true;
    }
}

Schritt 3: Rückgabe falls kein Wort gefunden wurde

Validierungsergebnis

return false;

Die Wortliste selbst wird beim Start aus einer Datei geladen und enthält 5-buchstabige Wörter:

Wörterliste laden

File file = new File("wordlist.txt");
            
// Wenn die Datei nicht im aktuellen Verzeichnis ist, versuche im ressource-Verzeichnis
if (!file.exists()) {
    file = new File("ressource/wordlist.txt");
}

BufferedReader reader = new BufferedReader(new FileReader(file));
String line;

while ((line = reader.readLine()) != null) {
    line = line.trim();
    if (line.length() == 5) {
        wordList.add(line);
    }
}

3 Übereinstimmungsprüfung

Nachdem festgestellt wurde, dass das eingegebene Wort gültig ist, wird überprüft, wie gut es mit dem Zielwort übereinstimmt. Dies geschieht in mehreren Schritten:

Schritt 1: Vorbereitung der Daten

Variablen initialisieren

eingabe = eingabe.toUpperCase();
zielwort = zielwort.toUpperCase();

int laenge = zielwort.length();

Color[] farben = new Color[laenge];

boolean[] zielwortBenutzt = new boolean[laenge];

Im ersten Schritt werden beide Wörter in Großbuchstaben umgewandelt, die Länge des Zielworts gespeichert und zwei Arrays erstellt:

  • farben: Speichert für jede Position die entsprechende Farbe
  • zielwortBenutzt: Verfolgt, welche Buchstaben des Zielworts bereits zugeordnet wurden

Schritt 2: Erste Überprüfung - Exakte Übereinstimmungen

Prüfung auf genaue Positionen

for (int i = 0; i < laenge; i++) {
    if (i < eingabe.length() && eingabe.charAt(i) == zielwort.charAt(i)) {
        farben[i] = Color.web("#6aaa64"); // Grün für richtige Position
        zielwortBenutzt[i] = true;
    } else {
        farben[i] = Color.web("#787c7e"); // Grau für falsche Buchstaben
    }
}

Schritt 3: Zweite Überprüfung - Buchstaben an falscher Position

Prüfung auf vorhandene Buchstaben

for (int i = 0; i < eingabe.length() && i < laenge; i++) {
    if (farben[i].equals(Color.web("#6aaa64"))) {
        continue; // Bereits als korrekt markiert
    }
    
    char buchstabe = eingabe.charAt(i);
    boolean gefunden = false;
    
    // Suche nach diesem Buchstaben im Zielwort
    for (int j = 0; j < laenge; j++) {
        if (!zielwortBenutzt[j] && zielwort.charAt(j) == buchstabe) {
            farben[i] = Color.web("#c9b458"); // Gelb
            zielwortBenutzt[j] = true;
            gefunden = true;
            break;
        }
    }
    
    if (!gefunden) {
        farben[i] = Color.web("#787c7e"); // Grau
    }
}

Schritt 4: Rückgabe des Farben-Arrays

Ergebnis zurückgeben

return farben;

4 Beispieldurchlauf

Um den Algorithmus besser zu verstehen, analysieren wir einen konkreten Fall:

  • Zielwort: "KATZE"
  • Eingabe: "TASSE"

Vorbereitung

Beispiel-Initialisierung

eingabe = "TASSE";
zielwort = "KATZE";
laenge = 5;
farben = [null, null, null, null, null];
zielwortBenutzt = [false, false, false, false, false];

Erster Durchgang: Exakte Übereinstimmungen

Position 0: 'T' in "TASSE" vs. 'K' in "KATZE"

Position 0 vergleichen

// Für i = 0:
if ('T' == 'K') { // falsch
    // Wird nicht ausgeführt
} else {
    farben[0] = Color.web("#787c7e"); // Grau
}

Position 1: 'A' in "TASSE" vs. 'A' in "KATZE"

Position 1 vergleichen

// Für i = 1:
if ('A' == 'A') { // wahr
    farben[1] = Color.web("#6aaa64"); // Grün
    zielwortBenutzt[1] = true;
}

Position 2: 'S' in "TASSE" vs. 'T' in "KATZE"

Position 2 vergleichen

// Für i = 2:
if ('S' == 'T') { // falsch
    // Wird nicht ausgeführt
} else {
    farben[2] = Color.web("#787c7e"); // Grau
}

Position 3: 'S' in "TASSE" vs. 'Z' in "KATZE"

Position 3 vergleichen

// Für i = 3:
if ('S' == 'Z') { // falsch
    // Wird nicht ausgeführt
} else {
    farben[3] = Color.web("#787c7e"); // Grau
}

Position 4: 'E' in "TASSE" vs. 'E' in "KATZE"

Position 4 vergleichen

// Für i = 4:
if ('E' == 'E') { // wahr
    farben[4] = Color.web("#6aaa64"); // Grün
    zielwortBenutzt[4] = true;
}

Nach dem ersten Durchgang:

farben = [grau, grün, grau, grau, grün];
zielwortBenutzt = [false, true, false, false, true];

Zweiter Durchgang: Buchstaben an falscher Position

Position 0: 'T' in "TASSE"

Position 0 vergleichen

// Für i = 0:
if (farben[0].equals(Color.web("#6aaa64"))) { // falsch, ist grau
    // Wird nicht ausgeführt
}

char buchstabe = 'T';
boolean gefunden = false;

// Suche nach 'T' im Zielwort "KATZE"
for (int j = 0; j < 5; j++) {
    // j = 0: 'K' != 'T' und nicht benutzt -> weiter
    // j = 1: 'A' != 'T' und bereits benutzt -> weiter
    // j = 2: 'T' == 'T' und nicht benutzt -> Treffer!
    if (!zielwortBenutzt[2] && zielwort.charAt(2) == 'T') {
        farben[0] = Color.web("#c9b458"); // Gelb
        zielwortBenutzt[2] = true;
        gefunden = true;
        break;
    }
}

Position 1: 'A' in "TASSE" - bereits grün, überspringen

Position 1 überspringen

// Für i = 1:
if (farben[1].equals(Color.web("#6aaa64"))) { // wahr, ist grün
    continue; // Überspringen
}

Position 2: 'S' in "TASSE"

Position 2 vergleichen

// Für i = 2:
if (farben[2].equals(Color.web("#6aaa64"))) { // falsch, ist grau
    // Wird nicht ausgeführt
}

char buchstabe = 'S';
boolean gefunden = false;

// Suche nach 'S' im Zielwort "KATZE"
for (int j = 0; j < 5; j++) {
    // j = 0: 'K' != 'S' und nicht benutzt -> weiter
    // j = 1: 'A' != 'S' und bereits benutzt -> weiter
    // j = 2: 'T' != 'S' und bereits benutzt -> weiter
    // j = 3: 'Z' != 'S' und nicht benutzt -> weiter
    // j = 4: 'E' != 'S' und bereits benutzt -> weiter
    // Keine Übereinstimmung gefunden
}

// Nach der Schleife:
if (!gefunden) {
    farben[2] = Color.web("#787c7e"); // Bleibt grau
}

Position 3: 'S' in "TASSE" - gleiche Prüfung wie Position 2, bleibt grau

Position 4: 'E' in "TASSE" - bereits grün, überspringen

Nach dem zweiten Durchgang:

farben = [gelb, grün, grau, grau, grün];

Ergebnis

Das endgültige Farben-Array zeigt dem Spieler:

  • 'T' an Position 0: gelb - Kommt im Zielwort vor, aber an anderer Stelle
  • 'A' an Position 1: grün - Korrekt an richtiger Position
  • 'S' an Position 2: grau - Kommt im Zielwort nicht vor
  • 'S' an Position 3: grau - Kommt im Zielwort nicht vor
  • 'E' an Position 4: grün - Korrekt an richtiger Position
